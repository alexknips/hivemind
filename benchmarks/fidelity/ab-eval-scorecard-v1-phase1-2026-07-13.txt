Key findings:

1. TOKEN-F1 TASKS (T1/T2/T4): Treatment consistently outperforms control.
   Structured JSON output reduces spurious tokens vs prose ADR, improving
   precision at equal recall. Biggest uplift: T2 rationale-reconstruction
   (+0.181) — the explicit evidence[] array in JSON avoids the prose
   verbosity (rationale text, consequences, etc.) that dilutes token overlap.

2. BINARY TASKS (T3/T5): No uplift in Phase 1. Both arms correctly detect
   contested/refuted status because the synthetic cases are explicit in both
   formats. The ADR documents use "Status: Contested" and "Refuted" headers
   that any model detects. This is a DESIGN LIMITATION of Phase 1 synthetic
   cases — they are too obvious.

3. T5 STALENESS: No Phase 1 uplift — EXPECTED. The synthetic write-batching
   case explicitly labels the hypothesis as refuted in the ADR. Real uplift
   from HiveMind's REFUTES link will show in Phase 1.5 with the Challenger
   corpus: there, control = scattered NASA/Thiokol docs where O-ring refutation
   is buried across files; treatment = explicit ASSUMES→REFUTES graph link.

4. NET UPLIFT +0.069 (9.7% relative): Positive direction but modest on Phase 1
   synthetic cases. The architecture is working: structured graph reduces
   retrieval noise. Full signal expected in Phase 1.5 (Challenger T5).
